## [1.0.0] - 2025-02-11

### Added
- **Message -> Safari 端到端能力**: Desktop Bridge 基础设施
  - `desktop.hotkey` - 发送快捷键（cmd+l, enter 等）

  - `desktop.dismissModal` - 关闭模态窗口
