# 任务单：/help 命令一致性审计

Issue: 0048  
Plan: aidocs/reviews/help-command-audit-260309.md

## 任务一句话

核对群聊 `/help` 与真实命令面是否一致,列出遗漏项、错误项与可简化点。

## 本轮范围

- 对照 `/help` 文案与真实路由/处理器命令
- 识别遗漏命令
- 识别参数描述过窄问题
- 给出最小修复和后续简化建议

## 非范围

- 不直接重命名命令
- 不合并多条入口链路
- 不擅自删除兼容命令

## 硬验收

1. 审计文档明确列出不一致项
2. 审计文档给出最小修复建议
3. 审计文档给出后续简化建议
