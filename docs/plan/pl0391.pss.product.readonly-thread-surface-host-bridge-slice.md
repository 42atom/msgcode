# PL0391：Readonly Thread Surface Host Bridge Slice

## 结论

下一刀不是继续加 UI 页面，而是把 `src/ui/main-window/readonly-thread-surface.ts` 接到最薄 Electron 宿主桥。

## 最小落点

1. main 只负责窗口和宿主整合。
2. preload 只暴露只读桥。
3. renderer 只消费：
   - `workspace-tree`
   - `thread`

## 禁止项

1. 不直接 import `src/runtime/*`
2. 不起新的本地 HTTP 服务
3. 不扩成多页 shell
4. 不回写 `ui-protype`

## 顺序

1. 落 preload 桥接口
2. 让 readonly thread surface 通过桥读取两个正式读面
3. 锁最小测试
