## 总目标

做出一个可安装、可分发、可长期使用的 AI appliance。

它要有：

- 完整 UI
- 稳定 CLI
- 清楚的 install / doctor 主链
- 可验证的任务闭环
- 先独立成立的 core appliance
- 之后再用 `wpkg` 去扩领域能力
