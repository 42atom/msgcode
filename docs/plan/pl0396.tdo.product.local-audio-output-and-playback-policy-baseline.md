# PL0396：Local Audio Output and Playback Policy Baseline

## 结论

先做本机音频输出，不做播放管理器。

产品层先固定：

- `output_channel`
- `playback_policy`

实现层只允许先做：

- `play`
- `stop`
- `status`

## 最小真相

### output_channel

- `system-default`
- `audio-device:<device-id>`
- `none`

### playback_policy

- `replace`
- `queue`
- `drop`

默认：

- TTS / agent 回复 = `replace`

## UI Position

- 首页：顶栏状态 pill 显示 `语音外放设备：...`
- 设置页：`general` 视图增加 `会话音频`
- 不在主窗口三栏里展开播放控制

## 顺序

1. 先收 settings / CLI / runtime 共用口径
2. 再开最薄实现单
3. 输入通道和 HomePod 后置
