# RP0349：workspace archive / restore 切片

## 结果

已增加两条正式 mutation：

- `msgcode appliance archive-workspace --workspace <labelOrPath> --json`
- `msgcode appliance restore-workspace --workspace <labelOrPath> --json`

## 主链

### archive

- 读取活跃 workspace 路径
- 计算 `.archive/<workspace>`
- 同名冲突先失败
- 成功后目录移动到 `.archive/`

### restore

- 读取 `.archive/<workspace>`
- 计算活跃根目录 `<workspace>`
- 同名冲突先失败
- 成功后目录移回活跃根

## 为什么这样够

- 不需要 archive 索引表
- 不需要 restore 编排层
- 不需要 UI 自己操作目录

## 失败口径

- 活跃 workspace 不存在：报错
- 归档 workspace 不存在：报错
- 同名冲突：报错
- 不做隐式重命名
