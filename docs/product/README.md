# 产品文档

本目录存放面向产品与交付的文档，分离“工程执行文档”和“产品叙事文档”。

## 目录结构

```text
docs/product/
  pitch.md                              # 产品定位/愿景/路线（对外叙事）
  msgcode-app-product-design_v0.1.md   # App 信息架构与页面草图
  msgcode-gui-ia-pages_v0.1.md         # GUI 信息架构图与页面清单（分阶段）
  msgcode-tool-loop-slo-v1.md          # Tool Loop 运行指标与降级标准
```

## 使用建议

- 面向开源用户：优先阅读仓库根 `README.md`（可运行、可排障）。
- 面向产品沟通：使用 `pitch.md` 作为统一对外叙事版本。
