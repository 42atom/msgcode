# gen skill

触发：图片生成、自拍、语音、音乐。

优先入口：`~/.config/msgcode/skills/gen/main.sh`

常用：
- `bash ~/.config/msgcode/skills/gen/main.sh image --prompt "赛博朋克猫" --json`
- `bash ~/.config/msgcode/skills/gen/main.sh selfie --ref ./AIDOCS/images/ref.png --json`
- `bash ~/.config/msgcode/skills/gen/main.sh tts --text "你好" --json`
- `bash ~/.config/msgcode/skills/gen/main.sh music --prompt "lofi" --json`
