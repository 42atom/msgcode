# Style Presets (3)

Use these presets when the user wants a style but does not specify a detailed prompt. Keep them as short prompt blocks.

## 1) Studio Product Photo
- Keywords: studio product photo, seamless backdrop, softbox lighting, clean reflections, sharp focus
- Composition: centered subject, eye-level, generous negative space
- Color: neutral palette, true-to-life materials
- Good for: ecommerce hero shots, catalog images

Preset prompt block:
"Studio product photo, seamless backdrop, softbox key light with subtle rim light, clean reflections, sharp focus, neutral palette, no text."

## 2) Cinematic Neon Noir
- Keywords: cinematic, neon noir, wet streets, high contrast, glow, bokeh
- Composition: dramatic framing, shallow depth of field
- Color: teal/magenta accents, deep shadows
- Good for: moody posters, nightlife scenes

Preset prompt block:
"Cinematic neon noir, wet street reflections, high contrast lighting, soft glow, shallow depth of field, teal and magenta accents, moody atmosphere, no text."

## 3) Clean Isometric Vector
- Keywords: clean isometric, vector illustration, crisp edges, minimal shading
- Composition: isometric perspective, tidy layout
- Color: limited palette, flat tones with subtle highlights
- Good for: product diagrams, app visuals

Preset prompt block:
"Clean isometric vector illustration, crisp edges, minimal shading, limited palette, tidy composition, flat tones with subtle highlights, no text."
