# Prompt Templates

Use these templates to keep prompts short, explicit, and reproducible.

## Generation Template
- Subject: <what to render>
- Style: <visual style or preset name>
- Composition: <framing, camera angle, subject placement>
- Lighting: <key light, mood, time of day>
- Background: <environment or backdrop>
- Quality: <high detail, clean edges, natural textures>
- Constraints: <"no text", "no watermark" if required>
- Negative: <only if user specifies>

Example:
"Subject: single ceramic teapot on a walnut table. Style: studio product photo. Composition: centered, eye-level. Lighting: softbox key light, gentle rim light. Background: neutral gray seamless. Quality: sharp focus, clean edges. Constraints: no text, no watermark."

## Edit Template
- Goal: <what must change>
- Preserve: <what must stay the same>
- Style shift: <optional style or preset>
- Constraints: <"keep composition", "keep color palette" if required>

Example:
"Goal: change the scene to a rainy night. Preserve: subject pose and framing. Style shift: cinematic neon noir. Constraints: keep facial features, no text."

## Describe Template
- Output: 1-2 sentence summary
- List: style, palette, lighting, notable defects or artifacts

Example:
"Describe the image in 1-2 sentences, then list style, palette, lighting, and any artifacts."
