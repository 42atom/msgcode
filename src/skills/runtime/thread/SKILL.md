# thread skill

触发：查看/切换会话线程，读取线程消息。

优先入口：`~/.config/msgcode/skills/thread/main.sh`

常用：
- `bash ~/.config/msgcode/skills/thread/main.sh list --json`
- `bash ~/.config/msgcode/skills/thread/main.sh active --json`
- `bash ~/.config/msgcode/skills/thread/main.sh messages <thread-id> --limit 20 --json`
- `bash ~/.config/msgcode/skills/thread/main.sh switch <thread-id> --json`
