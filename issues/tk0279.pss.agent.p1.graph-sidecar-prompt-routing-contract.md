# 任务：Graph Sidecar Prompt Contract

## Context

`0270` 已经有独立 sidecar CLI，但它还没有进入 agent 认知面。

当前最薄的接法不是接 runtime，而是先写进 prompt：

- 代码任务时优先尝试
- 失败时自然退回

## Goal / Non-Goals

### Goal

- 让模型知道何时优先尝试 `ts-graph-cli`
- 明确它只是上下文先验，不是控制层

### Non-Goals

- 不接 listener/runtime/tool bus
- 不新增新命令面
- 不把 sidecar 变成强依赖

## Plan

- [x] 在 coding lane 合同后补一段 graph sidecar 路由口径
- [x] 明确退回路径仍是 `rg / read_file / bash`
- [x] 补 prompt 文本回归锁

## Acceptance Criteria

1. prompt 里明确出现 `scripts/ts-graph-cli.ts context/impact`。
2. prompt 里明确 graph sidecar 只是优先尝试的上下文先验。
3. prompt 里明确失败时退回现役主链。
