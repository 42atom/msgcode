# 任务：shared front matter parser dedupe

## Context

当前至少这几处都在自己写 `parseSimpleFrontMatter()`：

- `thread-store`
- `workspace-thread-surface`
- `workspace-archive`

这不是功能问题，是维护债。

## Goal / Non-Goals

### Goal

- 抽一个最薄的共用 parser util
- 替换重复实现

### Non-Goals

- 不引入第三方 markdown 解析库
- 不扩成“完整 front matter 框架”
- 不顺手改 thread/archive 合同

## Plan

- [ ] 定一个最薄的 parser util 文件
- [ ] 替换 `thread-store`
- [ ] 替换 `workspace-thread-surface`
- [ ] 替换 `workspace-archive`
- [ ] 跑现有 thread/archive 读面验证

## Acceptance Criteria

1. 重复的 `parseSimpleFrontMatter()` 消失。
2. thread/archive 现有 front matter 读取行为不变。
3. 不引入新的 parser 层或第三方依赖。

## Notes

- 只收重复实现，不做功能增强。
- util 只服务当前 `key: value` 语义。
