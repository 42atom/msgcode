# 任务：wpkg 安装注册切片

## Context

当前仓库已经有：

- `WPKG.md`
- `packs.json` 读面
- `sites.json` 读面
- `appliance hall/sites` CLI

但还没有：

- 把 `.wpkg` 真正解压到工作区
- 校验 `manifest.json`
- 写入 `packs.json`
- 写入 `sites.json`

## Goal / Non-Goals

### Goal

- 增加最小 `wpkg install` CLI
- 解压合法 `.wpkg`
- 落到 `<workspace>/.msgcode/packs/user/<pack-id>/`
- 更新 `<workspace>/.msgcode/packs.json`
- 更新 `<workspace>/.msgcode/sites.json`

### Non-Goals

- 不实现升级
- 不实现卸载
- 不实现签名
- 不实现支付/市场
- 不实现 skill 自动发现

## Plan

- [ ] 定义最小安装入口
- [ ] 实现 manifest 校验
- [ ] 实现 pack 目录落盘
- [ ] 实现 packs.json / sites.json 更新
- [ ] 补黑盒测试
- [ ] 做一轮圆桌复核

## Acceptance Criteria

1. 合法 `.wpkg` 能安装到工作区并被 `appliance hall/sites` 看见。
2. 重复安装和坏包会返回真实错误，不静默覆盖。
3. 不新增包管理器，不新增数据库，不实现升级链。

## Notes

- 这张单只补最小安装注册链。
- `skill` 路径可先写入注册表留位，但当前不承诺 runtime 自动发现。
