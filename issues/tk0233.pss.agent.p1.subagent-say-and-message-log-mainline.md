# Goal

把 “子代理继续对话” 收成最薄主链：

- 有正式命令继续对运行中的子代理说一句话
- 每句话都落文件
- 不引入聊天平台，不新增中间控制面

## Non-Goals

- 不做多方聊天室
- 不做 inbox/mailbox 中台
- 不做 agent 自动互聊编排

## Result

- 已新增正式命令：
  - `msgcode subagent say <task-id> --message '...'`
- 已新增文件真相：
  - `<workspace>/.msgcode/subagents/<taskId>.messages.ndjson`
- 当前行为：
  - 向运行中的子代理继续发一句话
  - `--watch` 时记录本次回复
  - 每次消息都追加写入 message log
  - heartbeat 现已可消费子任务正文 `## Follow-up / ## Supervisor Message`
  - 对 running dispatch 只发送一次同内容 follow-up
  - 已写回 dispatch：
    - `lastSupervisorMessageHash`
    - `lastSupervisorMessageAt`
  - 子代理委派 prompt 已补最薄上下文：
    - 父任务摘要
    - 最近失败证据摘要
    - 现有 verify 口径
