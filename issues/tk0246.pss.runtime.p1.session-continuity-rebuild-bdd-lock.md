## Context

- session continuity 只能建立在 work continuity 之上。

## Goal / Non-Goals

- Goal:
  - 锁 session rebuild 依赖 work continuity
  - 锁 compaction 后仍能续跑
  - 锁 session 只是增益层
- Non-Goals:
  - 不把 session 当真相源
  - 不先做高级记忆召回

## Plan

- [x] 锁 session rebuild 依赖 work continuity
- [x] 锁 compaction 后仍可恢复
- [x] 锁 session 缺失时系统仍能工作

## Acceptance Criteria

1. 没有 session 也不丢活。
2. 有 session 时只会更顺，不会更假。
3. session rebuild 输出是最小恢复面，不是 transcript 全回放。
