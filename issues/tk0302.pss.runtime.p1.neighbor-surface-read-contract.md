# 任务：Neighbor 只读真相源合同

## Context

`0291` 已把邻居模块收成：

- 发现
- 联系人三态
- 异步邮箱
- 轻量可达探测

但当前仓库还没有一条稳定读面，能让 UI 或壳读取这些状态。

## Goal / Non-Goals

### Goal

- 给邻居页补一条最薄的只读主链
- 只读三份文件：
  - `.msgcode/neighbor/config.json`
  - `.msgcode/neighbor/neighbors.json`
  - `.msgcode/neighbor/mailbox.jsonl`
- 提供 `msgcode appliance neighbor --workspace ... --json`

### Non-Goals

- 不实现局域网发现
- 不实现 mailbox 写入
- 不实现 health 广播
- 不实现线程 owner / patch / review / handoff
- 不把邻居页做成第二任务系统

## Plan

- [x] 定义 neighbor surface 的最小字段
- [x] 实现 workspace-neighbor 读取器
- [x] 增加 appliance neighbor CLI
- [x] 补回归测试
- [x] 做一轮圆桌复核，吸收有价值约束

## Acceptance Criteria

1. 邻居模块开关只来自 `config.json`
2. 节点列表只来自 `neighbors.json`，mailbox 只补未读和最近消息
3. 邮箱记录只展示最近一段，不全量扫历史
4. 无文件时返回空态，不报假错
5. 非法文件会给 warning，不会静默吞掉

## Notes

- 这张单只补读面，不补任何网络或写入逻辑。
- 当前 UI 只需要节点头、节点列表和邮箱记录流；不要借这张单偷长线程系统。
