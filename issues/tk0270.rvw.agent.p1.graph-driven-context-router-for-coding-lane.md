# 任务：Graph Sidecar CLI

## Context

`0259 / 0277 / 0278` 已收口。

按单线程路线，下一步允许进入 `0270`，但只能做：

- 独立 sidecar CLI
- 不接主链
- 不长 manager / router policy / cache

## Goal / Non-Goals

### Goal

- 做一个本地 TS 文件图 sidecar CLI
- 支持 `context` / `impact` 两个最小命令面
- 输出机器可消费 JSON

### Non-Goals

- 不接入 `msgcode` runtime
- 不做 SQLite
- 不做 symbol 级精确图
- 不做 daemon/watch/UI

## Plan

- [x] 新增独立脚本 `scripts/ts-graph-cli.ts`
- [x] 先只建 TS 文件 import graph
- [x] 支持 `context` 输出：
  - `files`
  - `readOrder`
  - `tests`
  - `verify`
- [x] 支持 `impact` 输出：
  - `seedFiles`
  - `impactedFiles`
  - `relatedTests`
- [x] 补最小回归锁

## Acceptance Criteria

1. sidecar CLI 不依赖 `msgcode` 主链接线即可工作。
2. `context` 能根据 seed 文件给出最小读文件集合与 verify 建议。
3. `impact` 能根据反向 import 给出影响范围与相关测试。
