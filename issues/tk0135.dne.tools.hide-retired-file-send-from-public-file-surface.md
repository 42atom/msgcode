## Context

`help-docs --json` 已不再暴露 `file send`，但 `msgcode file --help` 仍把它列在公开命令面里。

这会形成新的分叉：

- 机器可读合同：`file send` 已 retired
- 人类公开 help：`file send` 仍像现役子命令

## Goal / Non-Goals

- Goal: 从 `file --help` 公开面移除 `send`
- Goal: 保留 `msgcode file send ...` direct invoke 兼容壳
- Non-Goals: 本轮不删除 retired `file send` 行为本身
