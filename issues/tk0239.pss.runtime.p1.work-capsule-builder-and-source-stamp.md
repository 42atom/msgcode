## Context

- wake 真正要带回来的不是一句提醒，而是“这份工作做到哪、下一步做什么”。

## Goal / Non-Goals

- Goal:
  - 定义 work capsule 最小结构
  - 组装 task-path 恢复输入
  - 给 capsule 加 source stamp
  - 明确轻路径不强制建 capsule
- Non-Goals:
  - 不持久化 capsule 为第二真相源
  - 不先做对话自然度优化
  - 不把 transcript 回放塞进 capsule

## Plan

- [x] 固定 work capsule 最小字段：`taskId/phase/checkpoint/nextAction/activeDispatch/vitalsSnapshot?`
- [x] 定义 source stamp 口径：底层 task/dispatch/checkpoint 变化即重建
- [x] 接 `taskId` wake 到 tk0205 的恢复协议
- [x] 补回归：无 capsule 不进入重路径正式执行

## Acceptance Criteria

1. 有 `taskId` 的 wake 可组装最小 work capsule。
2. capsule 至少包含 `taskId/phase/checkpoint/nextAction`。
3. rebuild 输出是最小胶囊，不是 transcript 全回放。
4. 底层真相变化后，旧 capsule 不得继续执行。

## Notes

- capsule 是恢复投影，不是持久化真相源。
