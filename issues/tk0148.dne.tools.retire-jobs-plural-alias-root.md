## Context

`msgcode job` 已是正式 canonical 入口，但 `msgcode jobs` 仍作为隐藏 alias root 存活，并在 CLI 归一化阶段静默映射到 `job`。

这不会执行第二套逻辑，但会继续保留两套根命名，和当前“公开面/执行面都尽量单一”的收口方向不一致。

## Goal / Non-Goals

- Goal: 退役 `jobs` 复数 alias 根入口
- Goal: direct invoke 返回显式迁移提示，统一引导到 `msgcode job`
- Goal: 更新相关测试与 changelog
- Non-Goals: 本轮不改 `msgcode job` 正式合同
- Non-Goals: 本轮不改 `src/cli/jobs.ts` 内部业务实现
