## Goal / Non-Goals

### Goals

- 在隔离目录完成 Patchright 安装与真实反检测验证。
- 验证 Patchright API 是否覆盖当前 browser runner 的核心 operation。

- 穷举 PinchTab 影响面，形成 Phase B 的改动清单。
- 输出 Phase A 报告与下一阶段 Plan 输入。

### Non-Goals

- 本轮不切换正式浏览器底座。
- 本轮不修改 `src/runners/browser-pinchtab.ts`、prompt 或启动链行为。
- 本轮不把 `patchright` 正式写入 `package.json`。
