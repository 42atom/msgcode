# 任务：workspace pack skill discovery prompt slice

## Context

当前已经有：

- `wpkg` 安装注册主链
- `<workspace>/.msgcode/packs.json`
- 主界面 pack 可见面

但主脑仍只知道：

- `~/.config/msgcode/skills/index.json`

这意味着：

- 已安装 `wpkg` 的站点主界面能看见
- 但其中的 `skills/**/SKILL.md` 还不会进入主脑发现面

## Goal / Non-Goals

- 让工作区已安装 pack 的 skill 能进入主脑 prompt 提示面
- 继续坚持：
  - 全局 skill 单源不变
  - 不新增工作区 skill 索引
  - 不同步到 `~/.config/msgcode/skills/index.json`
- 不做 skill 自动执行
- 不做市场/签名/支付
- 不做第二套技能中心
- 不做 UI 额外状态层

## Plan

- [x] 新增当前工作区 pack skill 读取器
- [x] 把当前工作区 pack skill 提示进 system prompt
- [x] 补回归锁，验证已安装 pack skill 会被提示
- [x] 确认不破坏全局 skill 单源

## Acceptance Criteria

1. system prompt 在当前工作区存在已安装 pack skill 时，能提示相应 `SKILL.md` 路径。
2. 提示面明确这些 skill 只属于当前工作区 pack，不同步到全局索引。
3. 没有 `packs.json` 或缺少 skill 文件时，不注入这段提示。

## Notes

- 这轮只补“提示面”，不补执行层。
- 继续坚持：agent 只有在任务确实涉及该 pack 能力时，才去读对应 `SKILL.md`。
- 不新增 `<workspace>/.msgcode/skills/index.json`。
