# 任务：Inbox Add CLI 切片

## Context

主界面输入框已经明确要做成 dumb terminal：

- 只负责投递
- 不自己维护消息真相
- 原始输入优先落文件

仓库里已经有：

- `.msgcode/inbox/`
- `createInboxRequest()`

所以这轮不该再发明 Web 专用消息总线。

## Goal / Non-Goals

### Goal

- 补 `msgcode inbox add`
- 让网页输入能正式落到 `.msgcode/inbox/*.new.*.md`
- 把 CLI 合同接进 `help-docs`

### Non-Goals

- 不补 inbox 消费器
- 不补 Web transport 主链
- 不改 listener
- 不做 UI optimistic bubble

## Plan

- [x] 新增 `msgcode inbox add`
- [x] 复用现有 `createInboxRequest()`
- [x] 接进 `help-docs`

1. `msgcode inbox add` 能把文本落成 inbox new 文件。
