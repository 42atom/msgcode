## Context

`src/tools/bus.ts` 中 `feishu_*` 五个分支仍在重复做三类事：

1. 读取 workspace config / 环境变量解析 `appId/appSecret`
2. 猜 `chatId` / `messageId`
3. 调具体 Feishu 工具并组装结果

这属于业务适配层，不是 Tool Bus 的纯网关职责。

## Goal / Non-Goals

- Goal: 将 `feishu_*` 的配置解析、上下文推断与调用编排移出 bus
- Goal: 保持工具集合、默认能力边界和返回结构不变
- Non-Goals: 本轮不改变 manifest 描述
- Non-Goals: 本轮不删 Feishu 的默认 chatId/messageId 语义
- Non-Goals: 本轮不重做 preview 合同
