## Goal / Non-Goals

- Goal:
  - 锁最小 capsule 字段
  - 锁 rebuild 先于模型恢复
  - 锁非 transcript 依赖
- Non-Goals:
  - 不锁 prompt 原句
  - 不引入第二份 capsule 真相源

## Plan

- [x] 锁 capsule 最小字段

2. 无 transcript 仍可重建 capsule。
